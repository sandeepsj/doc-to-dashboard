# Rust Basics

Learn the fundamentals of Rust — a fast, safe, and modern systems programming language. By the end of this tutorial, you'll understand Rust's ownership model, write programs with structs, enums, and error handling, and feel confident reading and writing basic Rust code.

## Prerequisites

- Familiarity with at least one programming language (Python, JavaScript, C, etc.)
- A terminal/command line you're comfortable using
- No prior Rust experience needed

## Roadmap

| # | Lesson | What You'll Learn |
|---|--------|-------------------|
| 1 | Getting Started | Install Rust, create a project with Cargo, run Hello World |
| 2 | Variables & Types | `let`, `mut`, shadowing, scalar and compound types |
| 3 | Functions & Control Flow | `fn`, `if/else`, `match`, loops |
| 4 | Ownership | Rust's ownership model — moves, scope, and memory safety |
| 5 | Borrowing & References | `&`, `&mut`, the borrowing rules |
| 6 | Structs & Methods | Define data structures and attach behavior |
| 7 | Enums & Pattern Matching | Enums, `match`, and `Option<T>` |
| 8 | Error Handling | `Result<T, E>`, the `?` operator, and `panic!` |
| 9 | Collections | `Vec<T>`, `HashMap`, and iteration |
| 10 | Traits & Generics | Shared behavior and type flexibility |
| 11 | What's Next | Summary, resources, and where to go from here |

## Estimated Time

**25-30 minutes** to read through the full tutorial. Budget extra time if you're coding along (recommended!).
